## [1.0.0] - 2025-12-18

### Added
- Initial extension specification
- Core signals: cue, fermata, attacca, vamp, caesura
- A2A-only signals: tacet, downbeat
- Performance parameters: tempo, dynamics, urgency
- Conditions support: hold_type, quality_threshold, repeat_until
- JSON Schema for validation
